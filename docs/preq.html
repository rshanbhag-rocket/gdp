Prerequisites to install Grafana Plugin Beta

You must complete certain tasks before you install Grafana Plugin Beta.

  • Installation of Tivoli® Enterprise Monitoring Server (TEMS) with PTF OA65625.
  • TEMS must be connected to the Zowe API Mediation Layer.
  • You must defined the ESM mapping between distributed identity and SAF userid.
